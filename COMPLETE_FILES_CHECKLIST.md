## 🚀 Installation Steps:

1. **Put all 13 files in your project root folder**

2. **Create required folders:**
```bash
mkdir fonts
mkdir images
```

3. **Download fonts** and put in `fonts/` folder:
   - Amiri-Bold.ttf
   - Amiri-Regular.ttf  
   - NotoSans-SemiBold.ttf
   - NotoSans-Regular.ttf

4. **Add background images** to `images/` folder (at least 1, recommended 50-100+)

5. **Install dependencies:**
```bash
pip install -r requirements.txt
```

6. **Launch the UI:**
```bash
streamlit run app.py
```

## 🎯 How It All Works:

1. **User opens Streamlit UI** (`streamlit run app.py`)
2. **Selects Surah** from dropdown or by number
3. **Uploads backgrounds** via drag-and-drop
4. **Adjusts colors/layout** with sliders and color pickers
5. **Clicks "Generate Videos"**
6. **UI calls** `python main.py` in background
7. **main.py:**
   - Reads `SETTINGS.py` 
   - Fetches Surah from API (or uses cache)
   - Calls `video_exporter.export_reels()` and `export_youtube()`
8. **video_exporter.py:**
   - Batches ayahs into correct lengths
   - Calls `clip_builder.build_clip()` for each batch
9. **clip_builder.py:**
   - Generates frames by calling `renderer.render_frame()` for each frame
   - Attaches audio
10. **Videos saved** to `videos/reels/` and `videos/youtube/`
